# Calculadora evolutiva en Java

Proyecto realizado en Java con Swing.

## Descripción
Esta aplicación es una calculadora básica que se va mejorando por fases.

## Fases
- Fase 1: operaciones básicas (+, -, *, /)
- Fase 2: mejoras visuales
- Fase 3: nuevas funcionalidades

## Tecnologías usadas
- Java
- Swing
- Eclipse
